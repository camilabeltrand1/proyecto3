Este proyecto tiene una landing page muy básica para seleccionar algunas de las operaciones disponibles: sumar, restar, multiplicar, dividir y sacar promedio. Además puede hacer conversiones de longitud y de tiempo. 

Los valores se ingresan mediante prompts y los resultados se muestran en consola.

Se permite ingresar valores decimales en todas las opciones excepto en la conversión de minutos a horas. Ahí la cantidad de minutos debe ser entera.

Al usar algunas de las opciones disponibles hay que ingresar los datos uno por uno (presionando "Ok"). Cuando ya se hayan ingresado todos los datos se presiona "Cancel" u "Ok" para finalizar y hacer la operación correspondiente.
